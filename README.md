# Cobiro website hub

Cobiro website, build with [GatsbyJS](https://gatsbyjs.org) and the scss-framework [baseguide](https://basegui.de).

## Requirements

Install Gatsby:

```
npm install -g gatsby-cli
```

Clone repo, cd in to project folder and then run:

```
npm install
```

## development

Run development server with hot reloading accessible by default at `localhost:8000`:

```
gatsby develop
```

## Production build

```
gatsby build
```

Serve production build locally:

```
gatsby serve
```
